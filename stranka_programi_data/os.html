Software a hardware

Programové vybavení (software) je nezbytné pro používání počítače (hardware). Dříve byly programy vytvářeny vždy pro konkrétní počítač, dnes existují systémové programy, které zajišťují běh aplikací.

Operační systémy (OS)

OS řídí hardware počítače. Jádro systému spravuje procesor, paměť, disky a další zařízení. Pro správnou funkci jsou potřebné ovladače (drivers). API umožňuje programátorům vytvářet aplikace pro daný OS.

Uživatelské rozhraní

Starší systémy používaly příkazový řádek (CLI), moderní OS mají grafické rozhraní (GUI) s okny a ikonami.

Historie OS

  • UNIX – víceuživatelský a víceúlohový systém (70. léta)
  • MS DOS a Mac OS – konkurenční systémy 80. let
  • Windows – grafické rozhraní pro PC
  • Linux – volně šiřitelný, populární na serverech

Typy OS podle zařízení

  • Servery – Windows Server, Linux, Unix
  • Osobní počítače – Windows, Linux, macOS
  • Mobilní zařízení – Android, iOS, Windows Phone
  • RTOS – systémy reálného času pro průmyslová a speciální zařízení
